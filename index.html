¿Qué es el CyberPi?

🔬 02 — Componentes

Sensores y componentes

El CyberPi incluye múltiples sensores que le permiten percibir el mundo a su alrededor. ¡Es como darle sentidos a una máquina!

💡

Sensor de Luz

Detecta la intensidad de la luz del ambiente. Útil para encender LEDs al oscurecer o controlar pantallas.

🎵

Sensor de Sonido

Mide el nivel de ruido. Puedes usarlo para que tu robot reaccione a aplausos o al volumen de la voz.

🌡️

Sensor de Temperatura

Registra la temperatura del ambiente. Ideal para estaciones meteorológicas o alarmas de calor.

📐

Giroscopio / IMU

Detecta movimiento, inclinación y rotación en 3 ejes. Permite controlar el robot moviéndolo en el aire.

🧲

Magnetómetro

Detecta campos magnéticos como una brújula digital. Sabe hacia dónde apunta el norte.

🎮

Joystick + Botones

Control físico integrado. Maneja tu robot, navega menús o controla juegos directamente desde el dispositivo.

📡

Infrarrojos (IR)

Emisor y receptor IR para comunicación entre dispositivos. ¡Dos CyberPi pueden "hablarse" sin cables!

🔊

Altavoz integrado

Reproduce tonos, música, alertas y efectos de sonido programables directamente desde el código.

💻 03 — Programación

¿Cómo se programa el CyberPi?

Usamos mBlock 5, un software que permite programar con bloques visuales (como Scratch) o con código Python real. ¡Tú eliges!

1

Descarga mBlock 5

Instala el software gratuito en tu computadora. Compatible con Windows, Mac y también funciona en el navegador.

2

Conecta el CyberPi

Usa el cable USB-C para conectar el CyberPi. mBlock lo detecta automáticamente y activa la comunicación.

3

Arrastra bloques o escribe código

En modo bloques: arrastra instrucciones como piezas de Lego. En modo Python: escribe código real línea por línea.

4

Sube el programa y prueba

Haz clic en "Subir" y tu código corre en el CyberPi. ¡Instantáneo! Prueba, falla, corrige y vuelve a intentar.

5

¡Comparte tu proyecto!

Guarda tu proyecto, muéstralo a tus compañeros y sube tus creaciones a la comunidad de mBlock.

📝 Ejemplo en Python / MicroPython
mi_proyecto.py — CyberPi
# 🤖 Mi primer programa con CyberPi
import cyberpi
import time

# Mostrar mensaje en pantalla
cyberpi.display.show_label(
    "¡Hola Mundo!", 16, "top_mid")

# Encender LEDs de colores
cyberpi.led.on("red", 0)
cyberpi.led.on("blue", 1)

# Reproducir sonido de bienvenida
cyberpi.audio.play("Ringtone")
time.sleep(2)

# Leer temperatura del sensor
temp = cyberpi.get_temperature()
cyberpi.display.show_label(
    f"Temp: {temp}°C", 14)

📖 05 — Vocabulario

Glosario de robótica

Palabras clave que debes conocer para hablar como un verdadero robotista.

Robot
Máquina programable capaz de realizar tareas automáticamente o siguiendo instrucciones de un programa.
Sensor
Dispositivo que detecta información del entorno (luz, temperatura, sonido) y la convierte en señal eléctrica.
Actuador
Componente que convierte señales eléctricas en acción física: motores, LEDs, parlantes, servo-motores.
Algoritmo
Conjunto de instrucciones ordenadas paso a paso que le dicen al robot exactamente qué hacer y cuándo.
Bucle (loop)
Instrucción que repite una acción muchas veces. Por ejemplo, hacer parpadear un LED 10 veces seguidas.
Variable
Espacio en la memoria del robot para guardar un dato que puede cambiar, como la temperatura actual.
Condicional (if)
Instrucción que toma decisiones: "Si la temperatura supera 30°, entonces enciende el ventilador."
ESP32
Microcontrolador de bajo costo con WiFi y Bluetooth integrado. Es el "cerebro" del CyberPi.
MicroPython
Versión simplificada del lenguaje Python diseñada para correr en microcontroladores como el ESP32 del CyberPi.
IoT
"Internet de las Cosas": conectar objetos cotidianos a internet para que envíen y reciban datos de forma remota.
mBlock 5
Software de programación por bloques y Python creado para CyberPi y otros dispositivos Makeblock.
LED RGB
Diodo emisor de luz que puede producir cualquier color mezclando Rojo (R), Verde (G) y Azul (B).
